네비게이션 메뉴 인디케이터 미니 프로젝트

1. 상단에 떠있는 네비게이션 바와 움직이는 인디케이터 구현

2. 메뉴 클릭 시 아이콘이 위로 올라가고 텍스트가 나타나며, 초록색 인디케이터가 부드럽게 이동

3. 기본 HTML, CSS, JavaScript 사용
   - CSS 변수(--clr) 활용
   - FontAwesome 아이콘 사용
   - CSS transform과 transition으로 부드러운 애니메이션 구현

4. 특징
   - 심플하고 모던한 디자인
   - 부드러운 애니메이션 효과
   - CSS 가상 선택자를 활용한 인디케이터 디자인
   - 클래스 토글을 통한 인터랙션 구현
